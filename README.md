# cocoapods-xccache

A CocoaPods plugin which cache pods with [zabel](https://github.com/WeijunDeng/Zabel) to speed compile.

## Installation

```shell
$ gem install cocoapods-xccache
```

## Usage

add the `plugin` to `Podfile`.

```ruby
plugin 'cocoapods-xccache'
```
